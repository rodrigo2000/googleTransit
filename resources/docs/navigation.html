Navigation

Huraga template come with two different types of navigation. Horizontal bar navigation on the top of the page and vertical navigation block on the left side. Both navigation design patterns are able to display second level child items.

Top horizontal navigation bar

Simply add your navigation content to an unordered list wrapped in nav tag.

Keep in mind that top navigation bar must be first item placed in header.

<header id="header" class="container">
	...
	<nav>
		<ul>
			<li><a href="#"> ... </a></li>
		</ul>
	</nav>
</header>

Side vertical navigation block

Navigation links are placed in simple HTML unordered list like navigation links in the horizontal navigation bar.

To add second level navigation, create a child list inside of parent list.

Note that you must include .no-submenu class for every link that does not contain second level navigation.

Add .current class to highlight active menu item.

<nav class="main-navigation" role="navigation">
	<ul>
		
		<!-- Simple link -->
		<li>
			<a href="index.html" class="no-submenu"> ... </a>
		</li>

		<!-- Link with second-level navigation -->
		<li>
			<a href="#"> ... </a>
			<ul>
				<li><a href="calendar.html"> ... </a></li>
			</ul>
		</li>

	</ul>
</nav>

Horizontal navigation bar widgets

Search query

Form tag with .nav-search class applied will turn text input into search bar widget.

Search query widget syntax
<form class="nav-search">
	<input type="text" placeholder="Search">
</form>

Vertical navigation block widgets

User profile

Place this predefined user profile pattern anywhere inside navigation block using .user-profile class.

Make sure to include all necessary tags like are shown in example.

User profile widget syntax
<div class="user-profile">
	<figure>
		<img src="avatar.jpg">
		<figcaption>
			<strong><a href="#"> ... </a></strong>
			<em><a href="#"> ... </a></em>
			<ul>
				<li><a class="btn" href="#"> ... </a></li>
				<li><a class="btn" href="#"> ... </a></li>
			</ul>
		</figcaption>
	</figure>
</div>

Side note

Use .side-note class with .side-note-container and .side-note-bottom blocks inside to create paper-like styled note widget.

Side note syntax
<div class="side-note">
	<div class="side-note-container">
		<h2> ... </h2>
		<p> ... </p>
	</div>
	<div class="side-note-bottom"></div>
</div>

Search query

Form tag with .side-search class applied will turn text input into search bar widget.

Search query widget syntax
<form class="side-search">
	<input type="text" placeholder="Search">
</form>

Balance

Balance widget is great way to pull out some numbers. Use .balance class with predefined content as you can see in the example.

Balance widget syntax
<div class="balance">
	<h2> ... </h2>
	<strong> ... </strong>
</div>

Badges

You can add badge for any link in navigation block by adding span tag with .badge class applied.

Badge widget syntax
<a>
	<!-- link content -->
	<span class="badge"> ... </span>
</a>
