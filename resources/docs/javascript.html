Javascript

Navigation

Navigaton.js is simple jQuery script to handle vertical navigation behaviour.

If you need to disable vertical navigation scroll fix you can easily do so by removing a particular part of code in navigation.js file.

Form gallery

jQuery gallery script to select multiple images via checkboxes.

$(document).ready(function() {
		
	/* Highlight gallery item when clicked */
	$('.form-gallery input[type="checkbox"]').click(function() {
		$(this).next('a').toggleClass('active');
	});
	$('.form-gallery input[type="checkbox"]:checked').next('a').addClass('active');
	
});

To-do list

When you use to-do list, do not forget to include this jQuery handler to fade out finished tasks.

$(document).ready(function() {
	
	$('.todo-block input[type="checkbox"]').click(function(){
		$(this).closest('tr').toggleClass('done');
	});
	$('.todo-block input[type="checkbox"]:checked').closest('tr').addClass('done');
	
});
