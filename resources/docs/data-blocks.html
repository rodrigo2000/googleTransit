Data blocks

Use standard Bootstrap grid classes with data-block cell to build your own layout. Each data block comes with optional components.

<article class="spanX data-block">
	<div class="data-container">
		<header>
			<h2>Headline</h2>
		</header>
		<section>
			Content
		</section>
		<footer>
			<!-- footer content -->
		</footer>
	</div>
</article>

Data block structure

First grid layer is built with .data-block cells. Second one works inside of data blocks. You can remove <footer> and <header> if needed or turn whole block into more complex widget.

Data block styles

You can choose from 3 different styles for each data block. Default style drops fancy papier-like shadows. For simpler look use class .decent or alternatively clean all styling with class .raw.

When you need to vertically align default and raw blocks headlines use class .fit to add top margin.
